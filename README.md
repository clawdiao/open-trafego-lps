## 🎨 Features Comuns

- Dark mode neon ( cyan + purple)
- Animações pesadas (GSAP, ScrollTrigger)
- 3D effects (Three.js ou CSS 3D transforms)
- Particle systems
- Video backgrounds
- Glassmorphism
- Parallax scrolling
- Magnetic buttons
- Real-time data demos
- testimonials com carrossel
- Pricing tables interativas
- Countdown timers (scarcity)
- Chatbot integrado (WhatsApp/Telegram)

## 🚀 Tech Stack

- HTML5 + Tailwind CSS
- Vanilla JS + GSAP
- Three.js (para 3D)
- Lottie animations (JSON)
- Fonts: Inter + Orbitron (tech feel)
